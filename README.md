# Nexora Framer Motion Site

Deploy on Cloudflare Pages with:
- Framework preset: Vite
- Build command: npm run build
- Build output directory: dist

Important files:
- index.html must be in the root
- package.json must be in the root
- src/main.jsx must contain the React code
- src/styles.css must contain the CSS

Add logo files here:
public/assets/nexora-logo.png
public/assets/nexora-icon.png
